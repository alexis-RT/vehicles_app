# vehicles_app
Aplicación web interactiva desarrollada con **Streamlit** para el análisis exploratorio
de un conjunto de datos de anuncios de venta de coches usados en Estados Unidos.

La aplicación permite visualizar la relación entre el precio, el kilometraje y otras
características de los vehículos mediante gráficos interactivos.

---

##  Funcionalidades

- Visualización de histogramas del kilometraje de los vehículos
- Gráficos de dispersión para analizar la relación entre:
  - Precio y kilometraje
- Segmentación de los datos por:
  - Condición del vehículo
  - Tipo de combustible
  - Tipo de transmisión
  - Tipo de vehículo
- Interfaz interactiva con controles dinámicos (checkbox y selectbox)

---

##  Tecnologías utilizadas

- **Python 3**
- **Pandas** – Manipulación y limpieza de datos
- **Streamlit** – Desarrollo de la aplicación web
- **Plotly Express** – Visualización de datos interactiva
